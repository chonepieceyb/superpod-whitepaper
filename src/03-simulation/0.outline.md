## 《AI大模型训练与推理建模仿真白皮书》 大纲

1. 建模仿真概述
   - 为什么需要负载级建模
   - 与时钟级 / 功能级的关系

2. 工作负载建模（Workload Modeling）
   - 仅定义应用需求（L0）
   - 不涉及系统与执行

3. 负载驱动的系统级仿真（System-level Simulation）
   - 以负载为输入
   - 以“训练 / 推理”为主要场景划分

   3.1 训练负载驱动的系统仿真
       - 计算 / 内存 / 通信在训练中的协同
       - CCL、依赖关系、MFU

   3.2 推理负载驱动的系统仿真
       - 请求到达过程
       - TTFT / TPOT / Tail latency

4. 资源级建模方法（Resource-level Modeling Methods）
   - 不再区分训练/推理
   - 作为“工具箱”

   4.1 计算与内存建模
       - 时钟级（GPGPU-SIM）
       - 功能级（Cmodel / SystemC）
       - 分析建模（Roofline）

   4.2 网络建模
       - 拓扑
       - 仿真器（ns-3 / garnet / OMNeT++）
       - 分析模型（bandwidth / delay）

5. 负载级仿真的关键挑战与应用
   - 仿真加速
   - 精度 vs 效率
   - 应用：系统设计 / 容量规划 / 策略评估
